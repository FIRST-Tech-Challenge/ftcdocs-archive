TensorFlow Archive
==================

TensorFlow was removed from the FTC SDK after the 2023 Centerstage game season.

*FIRST* no longer provides the servers that implement the FIRST Machine Learning toolchain **(FTC-ML)** that are required to train a TensorFlow model for use in the game challenges.
